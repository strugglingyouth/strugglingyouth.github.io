Q: 绑定远程分支，不用每次pull或push时指定

git config branch.master.remote origin
git config branch.master.merge refs/heads/master

Q:合并远程分支到当前分支

git merge origin/master 
# 或者
$ git rebase origin/master

Q:删除追踪状态的文件
